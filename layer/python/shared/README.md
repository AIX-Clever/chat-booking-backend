# Shared utilities

Código compartido entre todas las Lambdas.

## Contenido

- `dynamodb.py` - Helper para operaciones DynamoDB
- `utils.py` - Utilidades comunes
- `models.py` - Modelos de datos
- `constants.py` - Constantes del sistema

## Uso

Las Lambdas pueden importar este código usando Lambda Layers o copiando los archivos en cada función.
